## High risk indicator widget

### Objective

Create a widget that displays whether a patient is at high risk for a certain condition or not.

### Steps

1. Create a new enrollment plugin by using the `npm create @dhis2/app` command (or reuse a template)
2. Use the provided ids to fetch the data required for the widget. This could be either tracked entity, enrollment or event data.
3. Do some calculations to determine if the patient is at high risk for the condition. Examples can be if a pregnant woman has a high blood pressure or overdue.
4. Display the result in a widget.


Difficulty: Easy
