This repository contains all material and documentation for the plugin workshop in Amman, Jordan.

The workshop will be held on 15th-20th November 2025.

## Agenda

- Day 1 - Introduction & Environment Setup
- Day 2 - Form Field Plugins
- Day 3 - Enrollment & Dashboard Plugins
- Day 4 - Security & Server Middleware
- Day 5 - Pair Programming and Summary

## Repository Structure

```
├── day-1-initial-setup/          # Environment setup and Docker configuration
├── day-2-form-field-plugins/     # Form field plugin assignments and solutions
├── day-3-enrollment-and-dashboard-plugins/  # Enrollment and dashboard plugins
├── day-4-security-and-server-middleware/    # Security, routes, and middleware
├── day-5-pair-programming/       # Pair programming project ideas
├── advanced/                     # Advanced examples and middleware implementations
└── docker-compose.yml            # Docker configuration for DHIS2
```

## Getting Started

1. Clone this repository
2. Follow the guide in `day-1-initial-setup/guide.md` to set up your environment
3. Work through each day's assignments in order

## Prerequisites

- Docker Desktop installed
- Node.js v20+ and npm
- A code editor (VS Code recommended)
