# Assignment 3

## Objective

Render the fetched attributes in a @dhis2/ui DataTable. Data tables is used to display tabular data and are commonly used in DHIS2.

## Steps

1. Import the DataTable components from the @dhis2/ui library. You should import and use all of the components listed below.
2. Start with two columns: Attribute name and Attribute value.
3. Render the attributes as rows

# How to use the DataTable component


Link to the [documentation](https://developers.dhis2.org/demo/?path=/story/datatable--default)

```tsx
import {
    DataTable,
    DataTableHead,
    DataTableBody,
    DataTableRow,
```
