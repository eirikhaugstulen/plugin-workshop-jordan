# Objective
After finishing this assignment, you should be able to see the plugin with both the admin user and a restricted test user.


1. Open the day-4-enrollment-plugin folder
2. Run `npm install` to install the dependencies.
3. Run `npm start` to start the plugin
4. Add the plugin to the WHO RMNCH Program through the Tracker plugin configurator app.
5. Open a new browser window in incognito mode and log in with the restricted test user.
   1. Username: trackerAutoTestRestricted
   2. Password: Tracker123!
6. Open the WHO RMNCH Program
   1. The plugin should be visible for both the admin user and the restricted test user.
   2. It should display an error

# Restricted test user

The restricted test user is a reserved user that does not have a lot of access. It is a nice way to test how the system behaves for users with limited access.
