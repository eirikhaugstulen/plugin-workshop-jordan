# Assignment 1

## Objective

Clone the enrollment plugin template and display it in a DHIS2 tracker program.

## Steps

1. Clone the enrollment plugin template from https://github.com/eirikhaugstulen/jordan-enrollment-plugin-template
2. Install the dependencies by running `npm install` or `pnpm install`
3. Add the plugin to a tracker program by using the Tracker plugin configurator app

# Note

Do not change the Plugin.tsx file. Add all changes to the PluginContent.tsx file.
